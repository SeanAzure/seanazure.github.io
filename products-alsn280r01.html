2.8-inch RS485 Smart Display

Model No. AL-SN280R01

  • TFT Size: 2.8"
  • TFT Resolution: 320×240 dots
  • TFT View Direction: 80/80/80/80 (IPS)
  • Communication Interface: RS485/ UART
  • Touch Screen : Without Touch Panel
  • Brightness(cd/m²): 300
Description

The AzureLumi 2.8" RS485/UART Smart Display (Model: ALSN280R01) is designed for engineers and developers, combining advanced display technology with user-friendly development tools, allowing you to easily create various UI widgets.

This smart display is equipped with the standard 2.8” TFT LCD module. It features IPS panel technology with a wide viewing angle (80/80/80/80 degrees) and a resolution of 320×240 pixels. This configuration not only provides excellent display quality but also enhances the user interaction experience, making your applications more flexible and convenient.

Whether you are developing embedded systems or designing custom interfaces, the ALSN280R01 Smart Display offers reliable support to help you quickly realize your ideas and functionalities.

Features of the 2.8-inch RS485 smart display:
  • Input Voltage Range: 5V to 36V, with a typical voltage of 24V.
  • Self-Testing Boot Function.
  • Designed with STMicroelectronics' (ST) controller which supports ST development tools
  • Built-in 16MB External Flash Memory for storing fonts and object dictionary data.
  • Built-in Buzzer that can be controlled by the master device.
  • GPIO PIN Support for the usage of physical switches and indicators.
  • Built-in power polarity protection.
